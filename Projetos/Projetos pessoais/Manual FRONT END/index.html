Tipos de CSS

CSS In-line

     Podemos colocar estilos personalizados na mesma linha do código HTML, porém essa prática não é recomendada. Assim dificultando a manutenção do site para o programador e dificilmente será usada. Para manter boas práticas de código, devemos usar um estilo próprio de folha CSS.

CSS Interno

     Dentro da tag HEAD, iremos criar uma tag chamada "STYLE" e tudo que estiver dentro dessa tag será considerada CSS, assim separando o contéudo do estilo.
     A grande desvantagem de criar um style dentro da página html é que se eu tiver mais que uma página, precisarei estar fazendo a mesma coisa em todas as páginas, para manter boas práticas de código, devemos usar um estilo próprio de folha CSS.

CSS Externo

     Para criarmos uma folha externa de CSS, devemos utilizar o comando Link:css, fazendo com que crie uma folha própria de estilos e toda página de html que estiver linkado a folha de css, tornará para si os estilos personalizados dentro da folha de css.

Hierarquia de comandos de estilos

Existe uma hierarquia de estilos feitos pelo CSS, sendo:

  1. CSS In-line

          O CSS Inline terá prioridade máxima nos comandos de estilos, porque você está mudando o estilo daquela "linha".

  2. CSS Interno

        Na hierarquia de estilos, o css interno dentro da folha HTML irá sobrepor a folha de css externa porque você está mudando o estilo daquela folha.

  3. CSS Externo

        O CSS Externo terá baixa prioridade na linha de comando, porque você está usando uma "folha padrozinada" para todas suas páginas html.
