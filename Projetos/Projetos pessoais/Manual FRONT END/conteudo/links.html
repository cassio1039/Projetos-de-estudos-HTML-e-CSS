Usando links externos

Para criar um link você usa a tag A de âncora para criar algum link tanto INTERNO quanto EXTERNO

Não ESQUECER DO HTTPS para fazer que o link funcione

A tag (target="_blank" rel="external") usada sempre antes de fechar o >, serve para abrir o link em uma nova aba, não fazendo sobreescrever no seu site para não perder o usuário!

Usando link internos

Para avançar para a próxima página, usamos o envelopamento da palavra que queira deixar como link e criamos um link de âncora. Depois escolhemos a página em questão (detalhe: a página tem que estar dentro da própria pasta do projeto) CTRL + espaço e escolher a página!. Depois devemos OBRIGATÓRIAMENTE criar um "rel" para navegar entre as páginas INTERNAS com as palavras (next = próxima) e (prev = anterior)

Criando link para download

Exemplo:

  • Baixar o link em PDF
  • Baixar o link em .rar

Para criamos um link de download primeiramente devemos criar um link de âncora e no HREF colocar onde se encontra o arquivo e em seguida criar uma tag download="onde se encontra o arquivo/nome do arquivo e o nome da extensão" e logo em seguida usar o type="application + verificar no site a extensão correta "
