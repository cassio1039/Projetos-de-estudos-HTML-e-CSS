Principais formatações

Negrito / destaque

Nessa frase temos um termo em negrito usando a tag <b></b> do HTML. Mas usando B e depois a tecla TAB você abre a tag no VSCODE

Nesta frase, tenho um termo em destaque usando <strong></strong>(semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i></> (Não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em></em> (semântica).

Podemos criar também. um texto marcado usando a tag <mark></mark>.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído, com a tag <del></del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inseridocom a tag <ins></ins>para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u></u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, você usa a tag <sup></sup>

Texto subscrito

Para inserir coisas do tipo H20, você usa a tag <sub></sub>
