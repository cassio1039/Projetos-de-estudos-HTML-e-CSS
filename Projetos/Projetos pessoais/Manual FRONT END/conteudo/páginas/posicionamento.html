Posicionamento no CSS

Para fazer o posicionamento do background no css, temos algumas propriedades para serem levadas em conta e que nos ajuda a manter uma boa estrutura

background-position: Permite você posicionar a sua imagem de background em linha x, Y e por partes

background-repeat: Permite você escolher não repetir a imagem para ele não ficar duplicada

background-size: Permite você configurar o tamanho da imagem e existe duas principais tags que se usa

  • Contain: Serve pra usar a imagem em 100% da sua tela, porém ele não preenche todo o espaço delimitado no css
  • Cover:Serve para preencher sua imagem na tela inteira

background-attachment:

  • Scroll: Faz com o que o conteúdo navegue pelo site, mas a imagem de fundo acompanha o contéudo
  • Fixed: Faz com o que o contéudo do site navegue pelo site, mas a imagem do site fique fixa

Podemos também usar uma shorthand do background em que a ordem dele segue os seguintes parâmetros: color > image > position > repeat > size > attachment, porém vale lembrar que a especificação "size" não funciona e tem que colocar uma linha separada
