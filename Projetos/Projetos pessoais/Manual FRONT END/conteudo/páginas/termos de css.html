Termos usados em CSS

Aqui colocarei alguns termos bastante utilizados no CSS:

  • Height: Altura
  • Width: Largura
  • Padding: Preenchimento que fica entre a borda e o conteúdo
  • Margin: Margem do conteúdo após a borda
  • Border: Borda
  • Outline: Contorno
  • gap: Espaçar elementos dentro de um container-flex
