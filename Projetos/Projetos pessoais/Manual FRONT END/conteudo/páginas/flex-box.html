Flex-box

Para usarmos a flex-box temos algumas opções que podem ser usadas no pai (a div que comanda os itens) e no filho (a div que faz parte da estrutura do pai)

  • Container = pai:

  • display: flex
  • flex-direction

  • justify-content

    • Define o posicionamento pelo eixo X (horizontal(linha deitada)).

    • flex-start = Começa na esquerda para direita.
    • flex-end = Começa na direita para esquerda.
    • center = Deixa o conteúdo centralizado.
    • space-between = Distribui os elementos em tamanhos iguais.
    • space around = Coloca um espaço entre os elementos e em volta dos elementos.

  • flex-wrap

    Define se os itens devem quebrar ou não a linha.

    • nowrap = Valor padrão, não permite a quebra de linha.
    • wrap = Quebra a linha assim que um dos flex itens não puder mais ser compactado na linha atual.
    • wrap-reverse = Quebra a linha assim que um dos flex itens não puder mais ser compactado na linha atual e a quebra é na direção contrária, ou seja para a linha acima.
  • flex-flow

  • Align-items

    • Define o posicionamento pelo eixo Y (vertical (linha em pé)).

    • flex-start: Começa o posicionamento por cima.
    • flex-end: Começa o posicionamento por baixo.
    • center: Começa o posicionamento pelo centro.
    • stretch: Faz com o que os itens cresçam igualmente.
    • baseline: Alinha os itens de acordo com o tamanho da linha

  • Items = filho:

  • order

  • flex-grow

    • Faz com que os elementos se estiquem de maneiras iguais dentro da mesma caixa.

  • flex-shrink

    • Faz com que os elementos diminuam o tamanho de maneira igual

  • flex
  • align-self
