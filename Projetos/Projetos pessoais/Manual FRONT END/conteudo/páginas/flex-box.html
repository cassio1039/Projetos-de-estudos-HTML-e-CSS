Flex-box

Para usarmos a flex-box temos algumas opções que podem ser usadas no pai (a div que comanda os itens) e no filho (a div que faz parte da estrutura do pai)

  • Container = pai:

  • display: flex
  • flex-direction
  • justify-content
  • flex-wrap
  • flex-flow
  • align-items
  • align-content
  • flex item:

  • Items = filho:

  • order
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • align-self
