Trabalhando com listas

Listas ordenadas

Para criar uma lista ordenada (númerica) devemos utilizar a tag <ol></ol> e ainda dentro da tag ainda temos que colocar mudar o type para numeral.

Para criamos os itens da listas utilizamos a tag <li></li>.

Exemplo:

  1. Acordar
  2. Ligar para o joão
  3. Tomar café
  4. Escovar os dentes

Listas não ordenadas

Para criar uma lista não ordenada devemos utilizar a tag <ul></ul>.

Para criamos os itens da listas utilizamos a tag <li></li>.

Exemplo:

  • Pão
  • Leite
  • Tomate
  • Alface
  • Manteiga
  • Arroz
  • Feijão
